How I built this portfolio

When I started learning web development, I was tempted to use heavy frameworks like React. However, I realized that for a static portfolio, standard HTML and CSS is much faster and better for SEO.

The JavaScript Challenge

I wanted to create a toggle for the mobile menu. Here is the vanilla JavaScript function I wrote to handle the click event: 

const toggleMenu = () => {
    const nav = document.querySelector('.site-nav');
    
    nav.classList.toggle('is-open');
    
    if (nav.classList.contains('is-open')) {
        console.log("Menu opened!");
    }
}

This function simply toggles a CSS class, which keeps the JavaScript lightweight and the animation performance high (60fps).
